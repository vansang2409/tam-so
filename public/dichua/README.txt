GHI CHÚ

- Nên nén ảnh ~150–300 KB/tấm để web tải nhanh. Tỉ lệ 16:9 (vd 1920x1080).
- Prompt tạo ảnh: xem mục "Đi Chùa" trong WORK_LOG / trao đổi với AI.
